WooCommerce Payment Gateway for LinePay.

== Description ==

This is a WooCommerce payment gateway for LinePay.

It currently supports JPY (Japanese Yen) and USD (US Dollars) but more currencies will be added 
soon!

Follow these instructions to start your payment with LinePay!

1. Go to the following URL for registration:<br>
Korean - https://pay.line.me/ko_KR/apply/intro<br>
English - https://pay.line.me/en_US/apply/intro<br>
Japanese - https://pay.line.me/ja_JP/apply/intro<br>
2. Fill in your registration form.<br>
3. When you have received confirmation from LinePay, you can check your Channel ID and Channel 
Sceret in the following URL:<br>
Korean - https://pay.line.me/ko_KR/center/payment/interlockKey<br>
English - https://pay.line.me/en_US/center/payment/interlockKey<br>
Japanese - https://pay.line.me/ja_JP/center/payment/interlockKey<br>

All you need is your Channel ID and Channel Secret to start paying with LinePay!

<strong>PRO version is coming soon at http://planet8.co soon!</strong>
